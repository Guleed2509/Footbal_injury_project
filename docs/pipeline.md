# Injury Prediction Pipeline

This repository contains a full ML workflow for estimating football injury risk...

1) Data ingestion (`src/data_loader.py`)
2) Preprocessing (`src/preprocess.py`)
3) Feature engineering (`src/features.py`)
4) Model training (`src/train.py`)
5) Evaluation + metrics (`src/evaluate.py`)
6) Real-world inference (`src/predict.py`)
